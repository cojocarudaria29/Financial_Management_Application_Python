# 💹 Financial Management Application

O aplicație de gestionare financiară dezvoltată în **Python**, concepută pentru a automatiza procesarea și analiza tranzacțiilor economice. Proiectul utilizează fișiere structurate de tip CSV pentru a oferi o evidență clară a veniturilor și cheltuielilor.

---

## 🛠️ Caracteristici Principale

* **Procesare Batch:** Citirea și validarea automată a datelor din `date_financiare.csv`.
* **Analiză Financiară:** Calcularea automată a soldului total, a fluxului de numerar și a categoriilor de cheltuieli.
* **Structură Modulară:** Cod sursă organizat în `proiect.py` pentru a permite extinderea ușoară a funcționalităților.
* **Suport IDE:** Include configurații `.idea` pentru o integrare rapidă în ecosistemul JetBrains (PyCharm).

## 📂 Structura Proiectului

* `proiect.py` - Scriptul principal care execută logica de calcul și generarea rapoartelor.
* `date_financiare.csv` - Sursa de date care conține istoricul tranzacțiilor.
* `.idea/` - Fișiere de configurare pentru mediul de dezvoltare.
